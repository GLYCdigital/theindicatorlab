## What Is This?

The Indicator Lab reviews TradingView indicators so you don't have to waste time guessing which ones are worth installing. Every review is independent, hands-on, and straight to the point.

## How Reviews Work

Each indicator gets scored on:
- **Functionality** — Does it do what it promises?
- **Usability** — How easy is it to set up and tweak?
- **Value** — Is it worth the price (or the screen space if free)?
- **Originality** — Does it actually do something different?
